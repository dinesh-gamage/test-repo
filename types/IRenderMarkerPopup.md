# IRenderMarkerPopup




Render a popup for marker




```tsx
interface IRenderMarkerPopup {
    /**
     * content to show in popup
     */
    content: () => JSX.Element,
    /**
     * open the popup on load 
     */
    showOnLoad?: boolean
}
```

## Usage



```tsx
import {IRenderMarkerPopup} from 'uxp/components';
```

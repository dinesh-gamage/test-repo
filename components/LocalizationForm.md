# LocalizationForm




This component let's you to configure localisation messages for the enabled languages in iviva



## Installation



```tsx
import {LocalizationForm} from 'uxp/components';
```

## Examples



```tsx
<LocalizationForm
code: 'uxp-core.text.save'
/>
```

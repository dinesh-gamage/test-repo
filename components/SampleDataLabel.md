# SampleDataLabel





This component will be used to add a label to a widget to show that it's being rendered using sample data



## Installation



```tsx
import {SampleDataLabel} from 'uxp/components';
```

## Examples



```tsx
<SampleDataLabel
 show={true}
/>
```
